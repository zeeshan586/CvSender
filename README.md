# Automated CV Email Sender 🚀

A Node.js-based tool to automate job application emails with CV and cover letter attachments.

## ✨ Features
- Send bulk emails using SMTP (Gmail)
- Attach CV and cover letter dynamically
- Custom email templates
- Error logging (stores failed emails in JSON)
- Async email sending for better performance

## 🛠 Tech Stack
- Node.js
- Nodemailer
- File System (fs)

## 📂 How it works
1. Configure email credentials
2. Add recipient list
3. Run script to send emails automatically

## ⚠️ Note
Uses Gmail SMTP (App Password required)

## 🚀 Use Case
Useful for automating job applications and outreach campaigns.
